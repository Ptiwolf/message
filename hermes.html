MESSAGE (MSG) Utility Token

You've seen that an address has just bought a rug token and you want to warn the person.

But how do you say something to someone whose radix address you only know?

I've created this utility token, featuring Hermes the god of messengers, to answer that question: MESSAGE (MSG)

MESSAGE (MSG) on Radix Dashboard

How does it work?

The principle is simple: just send this token to the recipient's address, along with a memo containing your message. This message will appear in plain text in the transaction.

If you use https://launch.shardspace.app/ to send your token, you'll need to check the "Include Memo (quick send only)" box to compose your text.

If you use Radix Wallet's "Transfer" function, you have "+Add Message".

Once the token has been received by your recipient, he or she can reply in the same way, by sending the token back with a memo.
